О сервисе

Социальный граф основывается на данных о друзьях из социальной сети. Для отображения друзей используется force-directed graph, то есть граф, положение элементов которого обусловлено накладываемыми на них силами: пользователи отталкиваются друг от друга, а если пара пользователей дружит, то между ними добавляется сила притяжения.

Подобное поведение аналогично движению наэлектризованных отталкивающихся друг от друга бусинок, скреплённых натянутыми пружинками, если соответствующие бусинкам пользователи дружат. Благодаря этому, когда система приходит в равновесие, можно наблюдать разделение пользователей на группы (кластеры) с большим количеством дружеских связей внутри групп.

Для визуализации графа используется замечательная библиотека d3.js.

Справка

На больших социальных графах возможно подтормаживание. Для комфортного исследования графа дождитесь окончания загрузки и поставьте анимацию на паузу.

Поиск в графе можно выполнять по имени, фамилии, идентификатору, городу, ВУЗу и факультету пользователя. Найденные пользователи будут подсвечиваться синим цветом.

Ранжирование по лайкам – показываются друзья, которые лайкнули фотографии или записи на стене исследуемого пользователя. Чем больше лайков поставил друг – тем краснее его значок на социальном графе. Ранжирование по числу лайков позволяет выявить наиболее симпатизирующих пользователю друзей.

Политика конфиденциальности

Приложение не собирает и не хранит личные данные пользователей. Все необходимые данные загружаются приложением из социальной сети на устройство пользователя и не передаются на сторонние сервера.
